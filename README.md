# 🌾 AgriMind – AI-Powered Smart Farming Assistant

AgriMind is an intelligent agriculture assistant that helps farmers make data-driven decisions using Machine Learning, Weather Forecasting, and AI Agents.

It combines crop recommendation, 30-day weather forecasting, and irrigation planning into a unified AI-powered system.

Built for hackathons and scalable toward real-world AgriTech deployment.

---

## 🚀 Problem Statement

Farmers often struggle with:
- Choosing the right crop for their soil
- Planning irrigation efficiently
- Adapting to changing weather conditions

AgriMind provides AI-driven insights to support climate-smart agriculture and improve productivity.

---

## 🧠 Key Features

### 🌱 Crop Recommendation Engine
- Random Forest model trained on soil + climate dataset
- Inputs: N, P, K, pH, temperature, humidity, rainfall
- Outputs: Optimal crop recommendation with high accuracy

### 🌤 30-Day Weather Forecasting
- Fetches real-time weather data
- Predicts short-term climate patterns
- Supports better crop planning decisions

### 💧 Smart Irrigation Advisory
- Rule-based irrigation suggestions
- Adjusted using weather forecasts and crop type

### 🤖 AI Agent (LangChain + Groq)
- Intelligent orchestration of tools
- Natural language interaction
- Context-aware agricultural advice
